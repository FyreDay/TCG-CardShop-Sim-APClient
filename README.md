# TCG Card Shop Simulator - Archipelago Client Mod - NOW WITH UT SUPPORT
For the AP world see https://github.com/FyreDay/Archipelago-TCGCardShopSimulator/releases

## Setup
### Manual Setup

Mod TCG Card Shop Simulator with BepinEX with at least v5.4 Latest BepinEx is recommended

If you currently do not have TCG Card Shop Simulator modded, You have two options

1.Install BepinEX from either Nexus Mods or ThunderStore from the preconfigured mods available there

OR

2.Install BepinEX from https://github.com/BepInEx/BepInEx/releases/tag/v5.4.23.2

#### BepinEx Configuration

Once installed navigate to `TCG Card Shop Simulator\BepInEx\config` and open `BepInEx.cfg`

Under [ChainLoader] change HideManagerGameObject = false to HideManagerGameObject = true

#### Install Plugin

Extract the latest release into `TCG Card Shop Simulator\BepInEx\plugins`

### Thunderstore

This mod is also on thunderstore. Install Thunderstore on your computer and create a profile with AP Client

#### BepinEx Configuration

Once installed, Launch the game once to generate the config files

CLick 'Edit config' on the left

Under [ChainLoader] change HideManagerGameObject = false to HideManagerGameObject = true

## Connecting

If installed correctly you should see a window in the top left with fields to enter the IP:Port, password, and Slot. New Game and load game should be disabled. If you are ever able to see your save slots your installation has been done incorrectly.
To Connect, Enter the data and hit the connect button. If the `Not Connected` Text changes to `Connected` You have succesfully connected. You May now click New Game if starting, or Load game if you have previously played on this seed.
Your save slots will not be used and will be there after you uninstall the mod.

## How This Rando Works

### Goals
#### Reach Max Level
With this goal, you finish if you reach the capped max level
#### Collection Builder
With this goal, you finish when you collect a percentage of all the cards in the game
#### Sell Ghost Cards
With this goal, you are tryig to sell a target number of ghost cards and they are seeded as items throughout the multiworld

### Licenses and Level Caps

#### Item Licenses
Each item you can sell through the shop and tabletop shop have licenses you can find to unlock them in the multiworld. Each box of a item you sell is a check that will be sent. licenses also have level requirements so you need to be at level 10 or level 15 for them
to help with progression and leveling, 

#### Shop levels
Your shop levels with XP. However, your shop level will be capped until you are able to sell a target number of items every 5 levels. This means you will need to find Item licenses to level up.

### Workers
Workers can be unlocked by finding AP items, and there is an option to start with one unlocked at the very start.

### Play tables
Each game played in a format is a check. the formats are unlocked with items. 

### DIY
The DIY shop is reorganized and each item can be unlocked with AP items. Many of the same type are progressive unlocks from the same item. Below are all the progression items.

Item name: order -> of -> unlocks
"Progressive Shelf": Small Cabnet (Starting Item) -> small Shelf A -> shelf 2x2 -> small shelf B -> shelf
"Progressive Large Shelf": Long shelf -> LShapedShelf -> BoxShelf -> TetramonCircularShelf
"Progressive Pack Opener": Auto pack opener 1 -> 2 -> 3
"Progressive Card Table": Card table -> Vintage Card table
"Progressive Play Table": PlayTable -> BlackPlayTable -> WhitePlayTable -> TournamentPrizeShelf
"Progressive Card Display": Small Card display -> Medium Card display -> Large Card display 
"Progressive Personal Shelf":  Small Personal Shelf -> Medium Personal Shelf -> Large Personal Shelf
"Progressive Auto Scent": Autoscent -> autosent 1000 -> autoscent 5000
"Progressive Warehouse Shelf": Small Warehouse shelf -> Large Warehouse Shelf
"Progressive Wall Display Case": CardDisplayWallSmall -> CardDisplayWallBig -> CardDisplayWall3X2 -> CardDisplayWall1 -> CardDisplayWallBigWhite
"Progressive Card Projector": CardDisplayElectronicSmall -> CardDisplayElectronic -> CardDisplayElectronicLarge

### Auto Revovate
You can decide if you want the shop to expand automatically or if you have to pay money to expand
The warehouse is locked beheind the first "Warehouse Expansion" item.

### Cards
Card Checks are handled like achievements. Check your progress in the custom app in the phone!
Customers that want to trade will always have cards you dont own. Trade with them!

#### Opening Cards
Examples of easy Opening card checks are "Open 20 Common Tetramon Cards"

#### Card luck
There are items that will slowly make it so all cards have an equal chance. at 100 carc luck items, you will have a 1/6 for each card border and a 1/2 for foil!

#### Card sanity
If you really like the dopamine of opening cards, and the difficulty of opening a specific card, card sanity is here for you!
Each NEW card you get will send a check. There is an option to make it so foils are unique checks.
#### Selling Cards
Examples of easy Opening card checks are "Sell 20 Common Tetramon Cards"
#### Grading Cards
Examples of easy Opening card checks are "Grade 5 Common Tetramon Cards"

### Deathlink
You send a death to everyone else when a customer leaves your shop because it is stinky

When you recieve a death, randomly customers will leave. Watch out they may shoplift!

### Traps

stink_trap: Makes Customers stinky
poltergeist_trap: messes with lights
credit_card_failure_trap: makes it so no customers will use a credit card for a bit
decrease_card_luck_trap: lowers card luck
